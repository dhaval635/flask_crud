About Todo App

  • This Todo App is a simple CRUD (Create, Read, Update, Delete) web application created by Dhaval Suthar.

  • It allows users to manage their tasks efficiently. You can add new todos, update existing ones, and delete tasks you've completed.

  • Built with Flask, a web framework for Python, this app showcases the basic principles of web development.

  • Feel free to explore and use the app for organizing your tasks!

  • For more information or to contact the developer, you can visit Dhaval Suthar's GitHub profile: https://github.com/dhaval635
